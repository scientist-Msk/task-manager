# Task Manager CLI

Консольное приложение для управления задачами на Go.

## Возможности
- Добавление, удаление, выполнение задач
- История команд
- Разделение на активные и выполненные задачи

## Запуск
\```bash
go run main.go
\```

## Команды
- `add <заголовок> <текст>` - добавить задачу
- `del <заголовок>` - удалить
- `done <заголовок>` - отметить выполненной
- `list` - показать все задачи
- `events` - история команд
- `exit` - выход
